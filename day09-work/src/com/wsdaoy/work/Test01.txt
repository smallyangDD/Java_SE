1. 什么是接口，如何定义接口？
接口是java语言中的一种引用类型，是方法的集合。
使用interface关键字定义接口，其中可以定义抽象方法，默认方法，私有方法，静态方法。
public interface 接口名{
    抽象方法，默认方法，私有方法，静态方法。
}

2. 什么叫做多态，条件是什么？
一种食物的多种表现形式
条件：继承父类或者实现接口
        父类引用指向子类的对象
        子类或者实现类对方法的重写

3. 使用多态特性，带来了什么样的好处？
增强方法的扩展性和复用性。更好的参数的传递。

4. 使用多态特性，注意什么样的弊端？
不能调用子类的特有方法，必须向下转型。
